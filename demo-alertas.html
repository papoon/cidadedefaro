🎨 Sistema de Alertas Locais Não Intrusivos

Este é um sistema de alertas discreto e não intrusivo para informar utilizadores sobre eventos importantes, alterações relevantes ou situações excecionais.

Características:

  • ✅ Banner discreto que não bloqueia a navegação
  • ✅ Alertas categorizados por tipo (info, aviso, evento, sucesso, importante)
  • ✅ Possibilidade de fechar/dispensar alertas
  • ✅ Persistência opcional - alertas dispensados não aparecem novamente
  • ✅ Sem tracking ou notificações invasivas
  • ✅ Compatível com modo escuro
  • ✅ Totalmente acessível com teclado e leitores de ecrã

💻 Como Usar

Para mostrar um alerta na sua página, use a API JavaScript:

// Alerta simples window.localAlerts.show({ id: 'my-alert', type: 'info', title: 'Título do Alerta', message: 'Mensagem do alerta aqui.', dismissible: true, persistent: false }); // Alerta com auto-dismiss após 5 segundos window.localAlerts.show({ id: 'auto-dismiss-alert', type: 'success', message: 'Este alerta fecha automaticamente!', autoDismiss: 5000 }); // Tipos de alertas disponíveis: // - info: Informação geral // - warning: Avisos importantes // - event: Eventos e novidades // - success: Confirmações // - important: Alertas críticos

🎨 Casos de Uso

Exemplos de quando usar o sistema de alertas:

  • Eventos Grandes: "Festival de Verão em Faro de 15 a 20 de Julho"
  • Alterações de Serviço: "Horários de autocarros alterados durante o mês de Agosto"
  • Situações Excecionais: "Estrada para a Praia de Faro temporariamente fechada"
  • Atualizações Importantes: "Nova funcionalidade: Mapa interativo agora disponível"
  • Avisos Temporários: "Farmácia de serviço hoje: Farmácia Central"

✅ Critérios de Aceitação

  • ✅ Alertas não bloqueiam a navegação - aparecem no topo sem overlay
  • ✅ Fácil de desativar - botão de fechar claramente visível
  • ✅ Conteúdo claro - títulos e mensagens diretas
  • ✅ Sem tracking - nenhum dado é enviado para servidores externos
  • ✅ Sem notificações invasivas - tudo é controlado localmente
