Política de Privacidade

Como protegemos e tratamos os seus dados no Faro Formoso

📋 Introdução

O Faro Formoso respeita a sua privacidade e está comprometido em proteger os seus dados pessoais. Esta política de privacidade explica como tratamos as informações quando utiliza o nosso site.

Este site não recolhe, armazena ou processa dados pessoais identificáveis em servidores externos. Todas as funcionalidades que requerem armazenamento de dados são realizadas localmente no seu dispositivo.

🔍 Dados Recolhidos

Dados Armazenados Localmente

O Faro Formoso utiliza o armazenamento local do navegador (LocalStorage) para guardar preferências e melhorar a sua experiência. Estes dados nunca são enviados para servidores externos e incluem:

  • Preferências de idioma: Para lembrar se prefere ver o site em Português ou Inglês
  • Tema escolhido: Para lembrar se prefere o modo claro ou escuro
  • Favoritos: Lista de páginas e locais que marcou como favoritos
  • Cache de PWA: Páginas e recursos guardados para acesso offline

Dados Pessoais Não Recolhidos

O Faro Formoso NÃO recolhe ou armazena:

  • Nome, email ou informações de contacto
  • Endereço IP ou localização geográfica precisa
  • Informações de cartão de crédito ou pagamento
  • Histórico de navegação fora deste site
  • Cookies de rastreamento ou publicidade

💡 Como Utilizamos os Dados

Os dados armazenados localmente no seu dispositivo são utilizados exclusivamente para:

  1. Personalizar a experiência: Mostrar conteúdo no idioma preferido e tema escolhido
  2. Funcionalidade offline: Permitir acesso ao conteúdo sem ligação à internet
  3. Guardar favoritos: Manter uma lista dos seus locais e páginas preferidas

Importante: Como todos os dados são armazenados localmente no seu dispositivo, você tem controlo total sobre eles. Pode limpar estes dados a qualquer momento através das configurações do seu navegador.

🔌 Serviços Externos e APIs

O Faro Formoso integra informações de fontes públicas externas. Quando acede a estas informações, pode estar sujeito às políticas de privacidade dessas entidades:

  • GEO API PT: Dados geográficos e demográficos de Portugal
  • OpenStreetMap: Mapas interativos e dados geográficos
  • Fontes oficiais: Câmara Municipal de Faro, Portal do Cidadão

Estas integrações não partilham os seus dados pessoais. As consultas a APIs são feitas diretamente do seu dispositivo.

🍪 Cookies

Este site não utiliza cookies de rastreamento, cookies de publicidade ou cookies de terceiros. Utilizamos apenas o armazenamento local do navegador (LocalStorage e SessionStorage) para funcionalidades essenciais descritas acima.

🔒 Segurança

Implementamos medidas de segurança técnicas para proteger o site e os seus utilizadores:

  • HTTPS: Todas as comunicações são encriptadas
  • Content Security Policy (CSP): Proteção contra ataques de injeção de código
  • Sem backend: Não há servidor para armazenar ou comprometer dados
  • Código aberto: O código pode ser auditado publicamente no GitHub

⚖️ Os Seus Direitos

Como não recolhemos dados pessoais em servidores, você tem controlo total sobre todos os dados armazenados localmente:

  • Acesso: Todos os dados estão no seu dispositivo e podem ser inspecionados nas ferramentas de programador do navegador
  • Eliminação: Pode limpar todos os dados a qualquer momento nas configurações do navegador (limpar dados de navegação)
  • Portabilidade: Os favoritos podem ser exportados manualmente se desejar
  • Oposição: Pode optar por não usar as funcionalidades que armazenam dados localmente

🔗 Links para Sites Externos

O Faro Formoso contém links para sites externos (câmaras municipais, serviços públicos, etc.). Não somos responsáveis pelas políticas de privacidade desses sites. Recomendamos que leia as políticas de privacidade de qualquer site que visite.

👶 Privacidade de Menores

O Faro Formoso é um site de informação pública adequado para todas as idades. Não recolhemos intencionalmente dados de menores de 16 anos. Como não recolhemos dados pessoais, este site é seguro para uso por crianças sob supervisão de pais ou responsáveis.

📝 Alterações a Esta Política

Podemos atualizar esta política de privacidade ocasionalmente. Quaisquer alterações serão publicadas nesta página com a data de atualização. Recomendamos que reveja esta política periodicamente.
