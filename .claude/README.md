# .claude/

Configuration folder for Claude Code in this repository.

- `CLAUDE.md` — project instructions and coding guidelines that Claude Code loads automatically at the start of every session in this repo.
- `settings.json` — project-level Claude Code settings (permissions, etc.), shared via version control.
- `agents/` — custom subagent definitions scoped to this repo.
- `skills/` — custom project-scoped skills (`/<skill-name>`).
- `workflows/` — saved multi-agent orchestration scripts.
